# Lazuardy Mobile Backend

Backend API for online course Lazuardy Mobile Apps. build by Laravel 12.

---

## Tech Stack
- PHP 8.4.12
- Laravel 12
- MySQL
- Laravel sanctum

## Quick Start
1. composer install
2. cp .env.example .env
3. php artisan migrate --seed
4. php artisan serve
